# Помощник БФУ

Мобильное Android-приложение для студентов Балтийского федерального университета имени Иммануила Канта (БФУ), предназначенное для автоматизации учёта посещаемости и баллов по дисциплине «Физическая культура».

Приложение работает как клиент к существующей университетской системе учёта посещаемости и синхронизируется с ней через защищённое API.

---

## Основные возможности

### Авторизация

- Вход через централизованную систему БФУ (логин/пароль от `fc.kantiana.ru`)
- Проверка учётных данных через API университета
- Генерация сессионного токена (срок действия — 24 часа)
- Безопасное хранение токена на устройстве

---

### Журнал посещаемости

- Отображение общей доли посещений
- Отображение количества набранных баллов
- Круговая диаграмма статистики
- Подробная расшифровка начислений:
    - занятия
    - контрольные упражнения
    - LMS
    - спортивные мероприятия
    - физическая подготовка

#### Визуализация

- Горизонтальный скроллер месяцев
- Вертикальный список занятий выбранного месяца

**Цветовые статусы:**

- 🟢 — присутствие
- 🟠 — отсутствие по уважительной причине
- 🔴 — отсутствие
- ⚪ — занятие ожидается

---

### Онлайн / Офлайн режим

- Локальное кэширование данных
- Работа без интернет-соединения (просмотр сохранённой статистики)
- Ручная синхронизация с помощью жеста `pull-to-refresh`

---

### Персонализация

- Темы:
    - Светлая
    - Тёмная
    - Системная
- Языки интерфейса:
    - Русский
    - Английский

---

## Обновление данных

Для ручной синхронизации выполните жест **pull-to-refresh** на главном экране.

---

## Назначение проекта

Проект разработан как клиентское приложение к внутренней системе контроля посещаемости БФУ.

Предназначен для студентов
